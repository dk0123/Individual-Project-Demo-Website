To run the project, first the following commands need to be run: 
 source tf_env/bin/activate
 eval "$(pyenv init -)"
 python app.py

The following liabraries also need to be installed: and also the ones mentioned in requirements.py
Python(3.9.9)
Tensorflow(2.15.0)
OpenAI
Anthropic


Note: The age, gender and emotion detection models have not been included in this repository, since the files were too big for github. So, the code will not work as expected.
